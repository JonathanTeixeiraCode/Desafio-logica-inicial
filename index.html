História dos computadores

(Inicio)

Os computadores tiveram uma evolução muito grande desde de a metade do seculo XX, inicialmente feitos para calcular, e sendo maquinas gigantescas os primeiros computadores, sendo o ENIAC o de primeira geração de 1951 a 1959. Ja na segunda geração mesmo sendo de grandes proporções os computadores começaram a funcionar por meio de transistores isso em 1959 a 1965. Na terceira geração ja funcionavam com circuitos integrados, ficando menores no periodo de 1965 a 1975. A quarta geração ate os dias atuais eles estão cada vez menores e com muita capacidade de funcionamento.

O que são Clients?

(Inicio)

O cliente fornece para o usuário uma saída: a página renderizada que ele obteve do servidor (html). O cliente interpreta a entrada do usuário: o usuário entra com diferentes dados no cliente (e este eventualmente envia-os ao servidor).

O que são Servers?

(Inicio)

Servidores são computadores ou sistemas computacionais que atendem requisições de dispositivos clientes, através de uma ou mais redes (locais ou remotas), capazes de executar aplicações (programas), prover processamento e/ou capacidade de armazenamento de dados.

Como é uma estrutura básica de um HTML?

(Inicio)

A estrutura básica do HTML é composta das tags DOCTYPE, head e body. Obviamente, precisamos de mais tags para que a nossa página possua algum conteúdo, porém internamente o HTML é composto e executado por essas três principais tags.
